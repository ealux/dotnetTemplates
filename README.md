# Templates for dotnet

Provides templates for `dotnet new` command.

## Usage

To add template use in folder:
```
dotnet new install ./
```

To use template:
```bat
dotnet new mvvm
dotnet new ddd-mvc
```

## Templates

`mvvm`  - Model-View-ViewModel template.

`ddd-mvc` - Domain/DataBase/Tests/MVC infrastructure template.
